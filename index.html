MARK AI - Desktop Voice Assistant

MARK AI GUI

MARK AI Description

MARK AI is a voice-controlled desktop assistant for Windows PCs, developed using Python and Tkinter. It features integration with Google Gemini API , enabling smart & accurate responses.

The GUI is simple & user-friendly, designed for every user. MARK AI responds to voice commands & performs tasks efficiently, offering a smooth user experience.

The application size is around 1.5 GB due to various dependencies. This project showcases skills in Python, API integration, & desktop UI design.

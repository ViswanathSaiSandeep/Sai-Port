About The Project

Project Overview

  • Pramaha Translate is a real-time language translator for Windows PCs, developed using Python & Custom Tkinter.
  • Features integration with Microsoft Azure Cognitive Services for high-quality language translation.
  • The GUI is simple & user-friendly, designed for every user.
  • The app features 4 modes so that the user can select their preferred mode for their convenience.
  • The app is an optimised & light-weight application which enables faster responses.

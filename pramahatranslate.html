PRAMAHA TRANSLATE Description

Pramaha Translate is a real-time language translator for Windows PCs, developed using Python & Custom Tkinter. It features integration with GoogleTrans, which translates the text to other language.

The GUI is simple & user-friendly, designed for every user. The app features 4 modes so that the user can select their preferred mode for their convinience. The app is optimised & light-weight application which enables faster responses

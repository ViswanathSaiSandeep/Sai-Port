PARIS AI — Application Interface
About the Project

PARIS AI is a voice-controlled desktop assistant for Windows PCs, developed using Python and Tkinter. It features integration with Google Gemini API, enabling smart & accurate responses.

The GUI is simple & user-friendly, designed for every user. PARIS AI responds to voice commands & performs tasks efficiently, offering a smooth user experience.

The application size is around 1.5 GB due to various dependencies. This project showcases skills in Python, API integration, & desktop UI design.

Key Features & App Overview

Features Implemented

  • Greetings at Startup
  • Facial Recognition during Startup
  • Voice Interaction
  • Voice triggered WhatsApp messaging
  • Setting Alarms
  • Playing Music
  • Browsing Internet
  • Answering Queries through Gemini
  • Image Generation through Gemini
  • Capturing Images through Webcam
  • Controlling Computer through Voice — Opening Apps / Shutdown / Reboot / Battery Levels / Time & Date / Switch Tabs / Screenshots
  • Note taking through voice

Upcoming Features

  • Keyboard Interaction
  • Asking queries through webcam with Gemini Integration
  • Google Maps Integration
  • Sending Emails through Voice
  • Improved Facial Recognition
Known Issues
  • Longer Startup (30–60 sec)
  • Performance may be affected in noisy areas
  • Limited Security protocols
Release Status

PARIS AI is actively under development. Once all features are implemented and thoroughly tested for stability, the official release date will be announced. Regular updates will be posted here to keep you informed.
